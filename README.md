## Features

- **Auto-Accept**: Automatically clicks "Accept" for you.
- **Auto-Start**: Enabled by default for every editor when Antigravity opens.
- **Status Bar Toggle**: Click the status bar to turn on/off.
- **Keyboard Shortcut**: `Ctrl+Alt+A` (or `Cmd+Alt+A` on Mac).
